# 🎨 Image Generation Prompts — RetroSleuth: Case Files Detective

## 📐 STYLE GUIDE — Baca Ini Dulu

Semua gambar harus mengikuti style guide ini agar konsisten:

### Color Palette (Wajib Konsisten)
```
Primary:    #2d1b0e (dark brown — kayu jati, kulit)
Secondary:  #4a3728 (medium brown — kertas tua, karpet)
Accent:     #8b6914 (gold — jam tangan, kunci, detail mewah)
Highlight:  #c4a35a (light gold — pantulan cahaya, kristal)
Shadow:     #1a0f07 (deep shadow — sudut gelap, bayangan)
Skin:       #c4956a (warm skin tone — kulit Indonesia)
White:      #f5f0e8 (off-white — kertas, kemeja)
Black:      #0d0906 (near-black — pakaian formal, sepatu)
Red:        #8b2500 (dark red — karpet Persia, lipstik)
Green:      #1a4d1a (dark green — lampu meja, CRT)
```

### Lighting Style
```
- Single light source (lampu meja / lampu overhead)
- Dramatic chiaroscuro (kontras tinggi terang-gelap)
- Warm amber tone dari lampu minyak / lampu meja hijau
- Deep shadows di sudut ruangan
- No flat lighting — harus ada depth dan drama
```

### Film & Texture
```
- Heavy film grain (butiran film kasar khas 1970an)
- Slight vignette (pinggiran gelap)
- Warm color temperature (3200K — seperti lampu tungsten)
- Slight color shift ke amber/sepiA
- No digital sharpness — harus terlihat seperti foto analog
```

### Universal Negative Prompt
```
modern elements, digital artifacts, clean sharp edges, bright saturated colors, 
cartoon, anime, 3D render, CGI, plastic texture, LED lighting, neon colors, 
smartphone, computer screen, contemporary furniture, minimalist design, 
white background, studio lighting, flat lighting, oversaturated, HDR
```

### Universal Style Prefix (tambahkan di awal setiap prompt)
```
1970s Indonesian noir photograph, vintage analog camera, Kodak Tri-X 400 film grain,
dramatic chiaroscuro lighting, warm amber tones, muted color palette, 
cinematic composition, photorealistic, highly detailed
```

## 📸 FOTO TKP (Crime Scene)

### Style Konsisten untuk Semua Foto TKP:
```
- Camera angle: Slightly tilted (Dutch angle) untuk kesan dramatis
- Lighting: Flash photography harsh white light + ambient warm light
- Color: Desaturated dengan warm amber shift
- Evidence markers: Small yellow numbered cards (1-12) placed near evidence
- Ruler/scale: Sometimes visible for forensic reference
- Film grain: Very heavy (ISO 400 pushed to 800)
- Vignette: Strong
- Focus: Sharp on evidence, slightly soft on edges
- No people visible — empty crime scene
```

- Semua foto karakter: **sama background** (dark wood paneling), **sama lighting** (single warm source)

- Semua ikon: **sama palette** (brown/gold/dark), **sama border style** (1px gold)

- **Midjourney:** Tambahkan `--ar 4:5` (portrait) atau `--ar 4:3` (landscape) dan `--style raw`
- **Stable Diffusion:** CFG scale 7-9, Steps 30-50, DPM++ 2M Karras sampler
