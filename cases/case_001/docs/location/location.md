## 1.2 Pos Satpam (Gerbang Utama)

| Keterangan  | Detail                                              |
| ----------- | --------------------------------------------------- |
| **Lokasi**  | Tepat di samping gerbang utama, 20 meter dari Wisma |
| **Ukuran**  | 3m x 2.5m, bangunan kecil dari bata dan kayu        |
| **Luas**    | 2.5 m²                                              |
| **Petugas** | Tio (satpam) — shift malam 20.00-01.00              |

**Fasilitas Pos Satpam:**

- Meja kayu kecil dengan lampu meja minyak (listrik sering mati)
- Kursi kayu (agak reyot)
- Rak kecil berisi:
  - Buku log keamanan (evi_003)
  - Senter tua (baterai agak lemah)
  - Payung besar
  - Kotak makan siang
- Telepon mati (tidak tersambung ke Wisma)
- Jam dinding (baterai mati)

**Jarak Pandang:**

- Ke gerbang: 2 meter (langsung terlihat)
- Ke Wisma: 20 meter (terlihat jelas jika lampu di Wisma menyala)
- Ke taman belakang: TIDAK TERLIHAT (terhalang oleh Wisma)

**Sistem Keamanan:**

- Gerbang besi (2.5m) — terkunci pukul 21.00
- Tidak ada lampu sorot di taman belakang
- Tidak ada sistem alarm

**Pencatatan:**

- Tio mencatat setiap kendaraan yang masuk/keluar
- Catatan ditulis di buku log (terbuka, tanpa kunci)
- Buku log disimpan di laci meja (tidak terkunci)

**Catatan Penting:**

- Tio mengaku melihat Rahmat berjalan cepat ke arah taman belakang pada pukul 22.45
- Halaman 3 buku log hilang saat Tio periksa kembali pada pukul 02.30
