# Bukti — Catatan Panggilan Telepon (evi_020)

## Catatan Panggilan Keluar dari Klub Berkuda

---

| Keterangan | Detail |
| --- | --- |
| **ID Bukti** | `evi_020` |
| **Nama Bukti** | Catatan Panggilan Telepon Klub |
| **Sumber** | Catatan PABX Klub Berkuda Puncak |
| **Tipe** | Dokumen |
| **Tanggal** | 13 Juni 1979 |
| **Waktu Panggilan** | 21.30, 22.00 |
| **Nomor Penelepon** | Telepon Rendy di klub |
| **Nomor Tujuan** | Telepon Rumah Haryanto (Wisma Angker) |
| **Durasi** | 2 menit (21.30), 1 menit (22.00) |
| **Catatan** | Mencatat panggilan keluar dari Rendy ke Wisma Angker. Tidak ada panggilan masuk dari Wisma Angker ke nomor Rendy di klub setelah pukul 22.00. |
| **Relevansi** | Dapat membuat pemain curiga bahwa Rendy berbohong tentang menerima telepon dari Sari setelah pukul 22.00. Kenyataannya, Sari menelepon dari telepon umum. |
