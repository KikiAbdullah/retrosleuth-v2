# LAPORAN BARANG BUKTI FISIK

## KUNCI CADANGAN RUANG KERJA

**No. Barang Bukti:** BB-F/14/VI/1979/08  
**Tanggal Ditemukan:** 14 Juni 1979  
**Lokasi Penemuan:** Di bawah lipatan karpet di sudut dekat meja kerja

---

## DESKRIPSI FISIK

| Parameter  | Detail                                                  |
| ---------- | ------------------------------------------------------- |
| Jenis      | Anak kunci kuningan                                     |
| Panjang    | 4,5 cm                                                  |
| Gigi kunci | 5 gigi, pola tidak standar                              |
| Identitas  | Cocok untuk laci meja kerja dan laci lemari arsip       |
| Kondisi    | Aus di bagian gigi ketiga, menunjukkan sering digunakan |

---

## TEMUAN FORENSIK

| Pemeriksaan | Hasil                                                     |
| ----------- | --------------------------------------------------------- |
| Sidik jari  | Ditemukan sidik jari parsial (jari telunjuk dan ibu jari) |
| Pencocokan  | **Cocok dengan sidik jari Sari Wijaya**                   |
| Residu      | Tidak ditemukan residu racun                              |
| Serat       | Serat karpet Persia merah menempel di gagang              |

---

## KESIMPULAN

Kunci cadangan ini disembunyikan di bawah karpet dan hanya dapat diakses jika seseorang tahu lokasinya. Sidik jari Sari Wijaya menunjukkan bahwa ia adalah orang terakhir yang memegang kunci ini — kemungkinan untuk menyembunyikannya setelah digunakan.

---

_Kunci ini menjadi penghubung penting: ia membuka laci tersembunyi di meja kerja dan laci terkunci di lemari arsip — dua tempat penyimpanan bukti kunci._
