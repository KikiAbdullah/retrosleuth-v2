# LAPORAN ANALISIS BARANG BUKTI

## SERPIHAN GELAS BRANDY

**No. Barang Bukti:** BB-F/14/VI/1979/12  
**Tanggal Pengumpulan:** 14 Juni 1979  
**Lokasi:** Di atas karpet, 45 cm dari tangan kanan korban

---

## DESKRIPSI FISIK

| Parameter       | Detail                                                |
| --------------- | ----------------------------------------------------- |
| Jumlah serpihan | 8 pecahan besar + 15 pecahan kecil                    |
| Warna kaca      | Bening, kristal                                       |
| Ketebalan       | 2 mm                                                  |
| Jenis           | Gelas brandy kristal (cocok dengan set di meja minum) |

---

## TEMUAN FORENSIK

| Pemeriksaan   | Hasil                                                       |
| ------------- | ----------------------------------------------------------- |
| Residu cairan | **Positif Kalium Sianida (KCN)** — konsentrasi tinggi       |
| Alkohol       | Positif — brandy dengan kadar 40%                           |
| Sidik jari    | **Ditemukan 2 set sidik jari:**                             |
|               | 1. **Haryanto Wijaya** (korban) — pada bagian luar gelas    |
|               | 2. **Sari Wijaya** — pada bagian dalam gelas (tidak lazim!) |
| Bekas lipstik | **Positif** — warna merah muda (cocok dengan lipstik Sari)  |

---

## KESIMPULAN

Adanya sidik jari Sari pada **bagian dalam gelas** menunjukkan bahwa ia memegang gelas dari dalam — tindakan yang hanya terjadi jika seseorang menuangkan atau mencampurkan sesuatu ke dalam gelas. Residu sianida pada serpihan mengonfirmasi bahwa racun dicampur ke dalam brandy melalui gelas ini.

**Ini adalah bukti fisik terkuat yang menghubungkan Sari langsung dengan racun.**

---

_Serpihan disimpan dalam kantong plastik bersegel. Sidik jari telah difoto dan didokumentasikan._
